** Status Reference

- =8x= Note Off
- =9x= Note On
- =Ax= Key Pressure
- =Bx= Control Change
- =Cx= Program Change
- =Dx= Channel Pressure
- =Ex= Pitch Wheel
- =F0= System Exclusive
- =FF= Meta Event

*** Controllers

- =Bx 01 xx= Modulation (Vibrato)
- =Bx 07 xx= Channel Volume
- =Bx 0A xx= Channel Panning
- =Bx 0B xx= Expression
- =Bx 78 00= All Sound Off
- =Bx 79 00= Reset All Controllers
- =Bx 7B 00= All Notes Off
- =Bx 7E 00= Mono
- =Bx 7F 00= Poly

*** Meta Events

- =FF 2F 00=        End of Track
- =FF 51 03 tttttt= Set Tempo (microseconds per quarter-note = 60000000/BPM)
- =FF 7F 00=        Do nothing
