** Events

- =8x kk vv= Note Off
- =9x kk vv= Note On
- =Bx cc vv= Control Change
- =Cx    pp= Program Change
- =Ex ll hh= Pitch Wheel (center: =00 40=)
- =F0 .. F7= System Exclusive
- =FF .. ..= Meta Event

*** Controllers

- =Bx 00 vv= Bank Select (set /before/ program change)
- =Bx 01 vv= Modulation (Vibrato)
- =Bx 07 vv= Channel Volume
- =Bx 0A vv= Channel Panning
- =Bx 0B vv= Expression
- =Bx 78 00= All Sound Off
- =Bx 79 00= Reset All Controllers
- =Bx 7B 00= All Notes Off
- =Bx 7E 00= Mono
- =Bx 7F 00= Poly

https://en.wikipedia.org/wiki/General_MIDI#RPN

- =Bx 64 vv= RPN Low
  - =00= Pitch bend range
  - =01= Fine tune
  - =02= Coarse tune
- =Bx 65 vv= RPN High
- =Bx 06 vv= Data Entry

*** Meta Events

- =FF 2F 00=        End of Track
- =FF 51 03 tttttt= Set Tempo (microseconds per quarter-note = 60000000/BPM)
